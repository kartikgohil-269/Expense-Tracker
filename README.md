# 💰 Expense Tracker

A Full-Stack Expense Tracker built using **Node.js, Express.js, MongoDB, HTML, CSS, and JavaScript**. It helps users manage their daily expenses with a clean interface and powerful search and sorting features.

---

## 🚀 Features

- ➕ Add new expenses
- ✏️ Edit existing expenses
- ❌ Delete expenses
- 💾 Store data in MongoDB Atlas
- 🔍 Search by:
  - Title
  - Category
  - Amount Greater Than
  - Amount Less Than
  - Date
  - Month & Year
- 📊 Sort by:
  - Date (Newest → Oldest)
  - Date (Oldest → Newest)
  - Amount (High → Low)
  - Amount (Low → High)
- 💰 Displays total expenses
- 📱 Responsive user interface

## ⚙️ Installation

### 1. Clone the repository

```bash
git clone https://github.com/YOUR_USERNAME/Expense-Tracker.git
```

### 2. Go to the project folder

```bash
cd Expense-Tracker
```

### 3. Install dependencies

```bash
cd backend
npm install
```

### 4. Start the server

```bash
node server.js
```

### 5. Open the frontend

Open `frontend/index.html` in your browser.
